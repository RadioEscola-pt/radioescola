Amplificadores Operacionais

O que são Amplificadores Operacionais (Op-Amps)?

Os amplificadores operacionais são dispositivos eletrónicos que amplificam uma diferença de tensão entre dois terminais de entrada. Eles são amplamente usados em eletronica para realizar uma variedade de tarefas, como amplificação de sinais, filtragem, integração, diferenciação e muito mais.

Para efeito de cálculos é usado o modelo ideal de um opamp, algumas das suas características são:
  • Impedância de entrada infinita
  • Corrente de entrada nula
  • Ganho de tensão infinito
  • Impedânciade saida nula
  • Largura de banda inifinita
  • Tensão de desvio de entrada é nula
