Código de cores para resistências

As resistências eletrônicas possuem um código de cores que indica seu valor de resistência. Esse código é composto por faixas coloridas que representam os dígitos e multiplicadores.

Interpretação do Código de Cores

O código de cores de uma resistência é composto por faixas que representam dígitos e multiplicadores:

  • 1ª faixa: Primeiro dígito
  • 2ª faixa: Segundo dígito
  • 3ª faixa (Somente 5 Faixas): Terceiro dígito
  • 4ª faixa: Multiplicador
  • 5ª faixa: Tolerância

Exemplo de Resistência de 4 Faixas

Código de cores: 1234

  • 1ª faixa: 1 (Marrom)
  • 2ª faixa: 2 (Vermelho)
  • 3ª faixa: 10^2 = 100 (Preto)
  • 4ª faixa: ±5% (Dourado)

Isso resulta em um valor de resistência de 12 * 100 = 1200 ohms ou 1.2 kΩ com uma tolerância de ±5%.

Exemplo de Resistência de 5 Faixas

Código de cores: 12345

  • 1ª faixa: 1 (Marrom)
  • 2ª faixa: 2 (Vermelho)
  • 3ª faixa: 3 (Laranja)
  • 4ª faixa: 10^4 = 10000 (Amarelo)
  • 5ª faixa: ±5% (Dourado)

Isso resulta em um valor de resistência de 123 * 10000 = 1230000 ohms ou 1.23 MΩ com uma tolerância de ±5%.

Mnemônicos

Use Mnemônicos: Crie frases ou acrônimos usando as primeiras letras das cores para representar os números. Por exemplo, para as cores do código de resistores (preto, marrom, vermelho, laranja, amarelo, verde, azul, violeta, cinza, branco), você pode criar a frase "Parece Meu Velho Leão Andando Vagarosamente, Correndo Baixo" para representar os valores de 0 a 9.
